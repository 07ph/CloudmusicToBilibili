这是一个把网易云歌单信息转换成bilibili视频链接的工具
配合我写的油猴插件脚本，能批量收藏b站视频链接到自定义收藏夹

使用教程视频链接：还没录

- 写这个工具的原因是因为我wyy会员过期了，平常也没怎么用，想转b站听，配合ai写了两天，主要还是在调试，插件改自这位的哔哩哔哩自动添加收藏脚本https://github.com/AliubYiero/Yiero_WebScripts ，非常感谢！
- 工具的原理主要是
1. 从分享链接提取歌单ID，获取完整歌单信息
2. 从歌单数据中提取每首歌曲的名称和歌手信息
3. 对每首歌曲，使用b站搜索对应视频，收藏第一个视频（此时通常会因为b站反爬出现412错误而搜索失败）
4. 对搜索失败的歌曲进行重复搜索，通过加关键词（纯享、mv、交换前后文）并排列组合，提高成功率
- 因为收藏的是搜索到的第一个视频所以可能会收藏到不是你想要的那首歌，也有可能b站没有，所以你也可以根据需要，通过ai修改代码，修改关键词列表，或者调整搜索策略，来优化搜索效果
- 可能会有某些地方有bug，可以在我视频下面提出或者直接issue，但我不一定会修复，毕竟可能因为api失效各种原因导致无法搜索链接，我应该没那么多精力维护。
- 我最多转换过150首歌的歌单，更多我没做测试，越多的歌转换需要的时间越多，转换成功率也会变低，所以尽量不要转换大歌单。
- 批量收藏脚本也可以用来大量收藏b站视频，给链接就好了，至于链接什么的可以让ai整理。
